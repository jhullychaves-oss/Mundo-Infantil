# Validação de Dados Este projeto valida CPF, e-mail e telefone utilizando JavaScript. O usuário pode preencher os campos e, ao clicar no botão "Validar", o sistema informa se os dados são válidos ou inválidos. ## Funcionalidades - Validação de CPF. - Validação de E-mail. - Validação de Telefone.
